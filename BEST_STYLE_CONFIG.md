## 🔧 使用方法

### 在代码中调用
```python
from api_clients.jimeng_client import jimeng_client

# 生成单张图片
image_data = jimeng_client.generate_image(
    prompt="程序员小智在写代码，表情专注",
    style="cute",
    width=2048,
    height=2048
)

# 生成四格漫画
panels = jimeng_client.generate_four_panel_comic(
    prompts=[
        "程序员小智在写代码",
        "代码出现BUG",
        "BUG变成虫子",
        "惊蛰不宜写代码"
    ],
    style="cute"
)
```

## 📝 注意事项

1. **尺寸统一**: 必须使用2048×2048确保拼接时无空白
2. **间隔等待**: 每张图片之间等待5秒，避免API并发限制
3. **超时设置**: 单张图片最多等待5分钟
4. **错误处理**: 生成失败时自动创建灰色占位图

## 💡 优化建议

### 保持效果
- 使用 `style="cute"` 参数
- 保持 `scale=7.5` 的提示词相关度
- 使用随机种子 `seed=-1` 获得多样性

- 如需日漫风格：改用 `style="manga"`
- 如需简笔画：改用 `style="simple"`
- 如需更大尺寸：可尝试 3072×3072（文件会更大）
