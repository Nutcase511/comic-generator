# 🎨 公众号文章样式标准

**基于文件**: `output/preview.html`
**最后更新**: 2026-03-09
**状态**: ✅ 已验证可用

## 📋 样式标准定义

### 颜色主题
- 主色调: `#667eea` (紫色)
- 渐变色: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- 背景渐变: `linear-gradient(135deg, #f5f7fa 0%, #c3cfe2 100%)`
- 高亮背景: `linear-gradient(135deg, #ffeaa7 0%, #fdcb6e 100%)`

### 字体大小
- 主标题: 32px, font-weight: 700
- 章节标题: 22px, font-weight: 600
- 副标题: 14px
- 正文: 15px
- 代码: 14px

### 布局
- 容器最大宽度: 800px
- 四格漫画: 2x2网格布局 (grid-template-columns: 1fr 1fr)
- 圆角: 12px (容器), 10px (图片)
- 边框: 3px solid #667eea (章节标题)

### 文章结构（必须包含）

1. **标题区域**
   - 主标题 (h1)
   - 副标题

2. **🎨 四格漫画展示**
   - 2x2网格
   - 4个Panel卡片
   - 每个Panel包含: PANEL编号、图片、对话

3. **👥 角色介绍**
   - 渐变背景容器
   - 左边框 (5px solid #667eea)
   - 前缀符号: ▸

4. **📖 剧情详解**
   - 4个小节
   - 标题格式: "第一格：XXX"

5. **🔧 AI创作过程揭秘**
   - 📝 剧本生成提示词
   - 🎭 角色设计提示词
   - 💡 场景画面提示词

6. **💡 创作理念**
   - 3个段落
   - 加粗标题

7. **页脚**
   - 🤖 AI全自动生成
   - 技术栈标签 (白色背景, 紫色文字)

## 🔧 使用方法

### 自动化生成（推荐）
使用 `full_auto_workflow.py`:
```bash
python full_auto_workflow.py
```

这个脚本会自动：
1. 使用GLM-4生成完整剧本（包含visual_prompt）
2. 上传4张图片到微信素材库
3. 按照本标准生成HTML
4. 上传到微信公众号草稿箱

### 手动调用
在任何地方调用 `wechat_client.create_comic_article()` 都会自动使用此样式标准。
