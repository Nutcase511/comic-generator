# 火山方舟图像生成模型查找指南

## 🔍 模型可能的位置

### 位置1：模型广场
1. 访问火山方舟控制台
2. 点击左侧菜单 **"模型广场"**
3. 在顶部分类中点击 **"图片生成能力"** 或 **"视觉模型"**
4. 查找以下模型：
   - Doubao-Seedream 5.0
   - Doubao-Seedream 4.5
   - SeedEdit 3.0

### 位置2：在线推理
1. 点击左侧菜单 **"在线推理"**
2. 点击 **"创建推理接入点"**
3. 在模型选择器中：
   - 先选择 **模型类型**：图像生成
   - 再选择 **具体模型**：豆包Seedream系列

### 位置3：体验中心
1. 点击左侧菜单 **"体验中心"**
2. 选择 **"图像生成"** 分类
3. 可以直接体验模型，无需创建接入点

## ⚠️ 如果找不到豆包Seedream模型

### 可能的原因：

#### 1. 账号权限问题
- 图像生成可能需要单独申请权限
- 需要完成实名认证
- 可能需要企业认证

#### 2. 需要先购买/开通服务
- 某些模型需要先购买模型服务
- 可能有地域限制

#### 3. 使用即梦AI替代方案
如果火山方舟没有图像生成模型，可以直接使用即梦AI：
- 访问：https://jimeng.jianying.com/
- 下载客户端
- 使用API Key（从火山引擎控制台获取）

## 💡 推荐方案

### 方案A：查找并使用即梦AI API
即梦AI是字节跳动的图像生成服务，与豆包Seedream类似。

**获取API Key：**
1. 访问火山引擎控制台
2. 开通即梦AI服务
3. 获取Access Key

**调用方式：**
- 需要使用火山引擎的签名算法
- 比较复杂，但我们可以实现

### 方案B：使用其他图像生成API
如果即梦AI也无法使用，可以考虑：
- LiblibAI（免费额度）
- 堆友（阿里通义万相）
- 文心一格（百度）
